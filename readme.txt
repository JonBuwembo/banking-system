BANKING SYSTEM APPLICATION

This project is designed to simulate the basics of a real banking system, allowing a user to borrow, deposit, and withdraw money. The Application utilizes threading and Object Oriented Programming Concepts (OOP).

KEY FEATURES:

Deposit Money: Users can deposit funds into their account, with an option to automatically pay down any outstanding loans.
Withdraw Money: Users can withdraw funds, provided they have sufficient balance.
Take Out Loans: Users can take out loans with specified principal and interest rates.
Interest Accrual: The system automatically accrues interest on loans over time using threading.
Make Loan Payments: Users can make payments on their loans, reducing their balance over time.
View Account Balances: Users can view their current balance and outstanding loan amounts.
